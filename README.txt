README

Install the source under c:\pscf.

After downloading, your folder structre should look like this:

C:\
   pscf
      external
      msi-solution
      src
      system
      xls
      readme.rtf (this is the readme file that users see)

You will find the Visual Studio 2010 solution at C:\pscf\mis-solution\pscf.sln

If you want to begin by doing an end user installation you should run: 

      C:\pscf\msi-solution\msi-project\Release\setup.exe

Please send questions to marc@publicsectorcredit.org.
